# Card

<Subtitle>
    A card contains summary content and actions about a single subject.
    It can be used by itself or within a list, and is generally interactive.
</Subtitle>

## Usage

You can use the card with default content slot or combine header, footer for the complete look.

### Import

```jsx
import { DtCard } from '@dialpad/dialtone-vue';
```

## With header

A card with the header.

<Canvas>
    <Story id="components-card--with-header" />
</Canvas>

```html
 <dt-card class="d-w264">
    <template #header>
      <p>Header</p>
      <button class="d-btn d-btn--xs d-fc-black-400"><icon-menu-vertical /></button>
    </template>
    <template #content>
      <p class="d-fs14 d-fw-bold d-fc-black-800">Lorem ipsum</p>
      <p class="d-fs12">Lorem ipsum dolor sit amet, consectetur adipiscing elit.
      Donec fermentum molestie semper. Morbi finibus nulla turpis, nec molestie mi rutrum.</p>
    </template>
</dt-card>
```

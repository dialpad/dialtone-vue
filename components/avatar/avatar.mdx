## Usage

### Import

```jsx
import { DtAvatar } from '@dialpad/dialtone-vue';
```

### Valid Slot Markup

There are three valid types of markup you can pass into an avatar slot. `<img>` for custom images,
`<svg>` for icons, and raw text for initials.

#### Image

`src` and `alt` are required attributes for image avatars

```jsx
<dt-avatar>
  <img src="/path/to/image" alt="image alt text">
</dt-avatar>
```

#### Svg (with dialtone icon)

```jsx
<dt-avatar>
  <icon-dialpad-glyph/>
</dt-avatar>
```

#### Svg

```jsx
<dt-avatar>
  <svg width="24" height="24" viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg">
    <path d="M5 10H9V15H15V10H19L12 3L5 10Z" fill="black"/>
    <path d="M5 20V18H19V20H5Z" fill="black"/>
  </svg>
</dt-avatar>
```

#### With Raw Text (initials)

```jsx
<dt-avatar>
  DP
</dt-avatar>
```

### Seeded

You may use a seed to make the randomly generated colors be the same every time for that seed. This is useful
if you want users to always have the same color, just pass in their unique user ID as the seed.

```jsx
// Will always show the same colors for this seed.
<dt-avatar
  seed="X5G3D7D3DS3WL7"
/>
```

### With Size

```jsx
<dt-avatar
  size="xs | sm | md | lg | xl"
/>
```

# KeyboardShortcut

<Subtitle>
    Keyboard shortcut displays the combinations of keys that provide an quicker way to do something that you'd typically
    do with a mouse.
</Subtitle>

## Base Style

Keyboard shortcut is made from an outline border with text and or keyboard symbols inside to represent the key
combination. If there are more than 1 key, it is connected with a “+” in between.

## Usage

- Commands shown should always be device specific (different systems may require a different shortcut combination).
- Commonly used in list item, tooltips.
- Don’t use actual “+” key as a shortcut.
- Avoid using browser or operating system shortcuts, Control+P is print for example.

### Import

```jsx
import { DtKeyboardShortcut } from '@dialpad/dialtone-vue';
```

### Shortcut size variation

```html
<dt-keyboard-shortcut
  class="d-fs16"
  shortcut="{cmd} Y"
/>
```

### Accessible shortcut

```html
<dt-keyboard-shortcut
  :aria-label="$i18n('Press Meta-Right Arrow-Left Arrow-Y')"
  tabindex="0"
  shortcut="{cmd} {arrow-right} {arrow-left} Y"
/>
```

### Accessibility

- You can use `aria-label` and `tab-index` to make this component accessible.
- it's
[`aria-keyshortcuts`](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Attributes/aria-keyshortcuts)
attribute, which indicates keyboard shortcuts that an author has implemented to activate or give focus to an element.
- Abbreviations / symbols should be read out in full for voiceover / screen readers.

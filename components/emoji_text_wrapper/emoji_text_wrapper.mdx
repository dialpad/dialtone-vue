## Usage

### Import

```jsx
import { DtEmojiTextWrapper } from '@dialpad/dialtone-vue';
```

### Limitations

Currently, this wrapper component **can't handle Vue components**, make sure to wrap only native HTML elements and text.

### With Shortcode

```html
<dt-emoji-text-wrapper>
  Example text with bigger :smile: emoji.
</dt-emoji-text-wrapper>
```

### With Unicode Character

```html
<dt-emoji-text-wrapper>
  Example text with bigger üòÑ emoji.
</dt-emoji-text-wrapper>
```

### With Nested Structure

```html
<dt-emoji-text-wrapper>
  <div>
    <button class="d-btn d-btn--primary">Example button :cry:</button>
    <div>
      <span>Another example üßê</span>
    </div>
  </div>
</dt-emoji-text-wrapper>
```

### With Emoji Size

```html
<dt-emoji-text-wrapper size="d-svg--48">
  ...
</dt-emoji-text-wrapper>
```

## With Custom Emoji

[Set up the custom emoji](?path=/docs/components-emoji--custom-emoji#custom-emoji) in order to render with this component.

```html
<template>
  <dt-emoji-text-wrapper>
    custom emojis :octocat: :shipit:
  </dt-emoji-text-wrapper>
</template>

<script>
import { DtEmojiTextWrapper } from '@dialpad/dialtone-vue';

export default {
  components: { DtEmojiTextWrapper },
}
</script>
```

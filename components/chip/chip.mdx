## Usage

Default behaviour is interactive, has hover/focus/active states. When interactive the chip will render as a `<button>`
element.

Default content text is aria-label unless overridden by passed in aria-label.

```js
import { DtChip } from '@dialpad/dialtone-vue';
```

```html
<dt-chip :close-button-props="{ ariaLabel: $i18n('close') }">
  Chip
</dt-chip>
```

## Non Interactive

The chip is not focusable or clickable. If not interactive the chip will render as a `<span>` element.

<Canvas>
  <Story id="components-chip--non-interactive" />
</Canvas>

```html
<dt-chip
  :interactive="false"
  :hide-close="true"
/>
```

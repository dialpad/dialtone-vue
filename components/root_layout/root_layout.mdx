# DtRootLayout

<Subtitle>
  A Root Layout provides a standardized group of containers to hold content.
</Subtitle>

## Base Style

A root layout is used by providing content to any of the provided slots.
It is intended mainly for use at the root level of design.

## Usage

### Import

```jsx
import { DtRootLayout } from '@dialpad/dialtone-vue';
```

### Default

```html
<template>
  <dt-root-layout>
    <template #header>
      <div>Header</div>
    </template>
    <template #sidebar>
      <div>
      <div>Sidebar item 1</div>
      <div>Sidebar item 2</div>
      <div>Sidebar item 3</div>
      </div>
    </template>
    <template>
      <div>Content</div>
    </template>
    <template #footer>
      <div>Footer</div>
    </template>
  </dt-root-layout>
</template>
```

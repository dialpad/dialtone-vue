# DtRootLayout

<Subtitle>
  A root layout provides a standardized group of containers to display content at the root level.
</Subtitle>

## Base Style

A root layout consists of a header, body, sidebar and footer. Content passed into the slots will be displayed in
the respective area. The sidebar is designed to be responsive and will reposition above the body when the content
reaches a relative breakpoint.

## Usage

### Import

```jsx
import { DtRootLayout } from '@dialpad/dialtone-vue';
```

```html
<template>
  <dt-root-layout>
    <template #header>
      <div>Header</div>
    </template>
    <template #sidebar>
      <div>
        <div>Sidebar item 1</div>
        <div>Sidebar item 2</div>
        <div>Sidebar item 3</div>
      </div>
    </template>
    <template>
      <div>Content</div>
    </template>
    <template #footer>
      <div>Footer</div>
    </template>
  </dt-root-layout>
</template>
```

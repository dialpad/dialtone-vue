## Usage

### Import

```jsx
import { DtSkeleton } from '@dialpad/dialtone-vue';
```

### Skeleton text label

```html
<dt-skeleton />
```

### Skeleton text heading

```html
<dt-skeleton arial-label="loading"  :textOption="{
  type: 'heading'
}"/>
```

### Skeleton paragraph

Random widths

```html
<dt-skeleton arial-label="loading"  :paragraphOption="{
          rows: 5,
          minWidth: 50,
          maxWidth: 80,
          randomWidth: true,
          offset: 0.7,
}"/>
```

Fixed widths

```html
<dt-skeleton arial-label="loading"  :paragraphOption="{
          rows: 5,
          offset: 0.7,
          width: ['50%', '40%']
}"/>
```

### Skeleton list item

```html
<dt-skeleton arial-label="loading"  :listItemOption="{
  paragraphs: {
          rows: 5,
          minWidth: 50,
          maxWidth: 80,
          randomWidth: true,
          offset: 0.7,
  },
  shape: 'rectangle'
}"/>
```

### Skeleton shape

```html
<dt-skeleton arial-label="loading" :shapeOption="{
  shape: 'square',
  size: 'lg'
}"/>
```

## Accessibility

If you need fully accessible skeleton (for a reason to load article for example)
(it's not grid loading case), probably you will want to add aria-live="polite" aria-busy="false" to content html parent element
to make it visible for screenreader content is loaded.

[Aria-busy](https://accessibilityresources.org/aria-busy)

[ARIA22](https://www.w3.org/WAI/WCAG21/Techniques/aria/ARIA22)

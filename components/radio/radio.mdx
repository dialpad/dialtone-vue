# Radios

<Subtitle>
  Radios are control elements that allow the user to make a single selection. They are typically used in a Radio Group
  which allows the user to make a selection from a list of options.
</Subtitle>

## Base Style

Radios are typically paired with a label which identifies the option. There are times when they can be used with a
description which describes the option.

## Usage

### Import

```jsx
import { DtRadio, VALIDATION_MESSAGE_TYPES } from '@dialpad/handset';
```

### With Label Prop

```vue
<dt-radio
  value="optionValue"
  label="My Label"
/>
```

### With Slotted Label

```vue
<dt-radio value="optionValue">
  My Label
</dt-radio>
```

### With Description

```vue
<dt-radio
  value="optionValue"
  description="My Description"
>
  My Label
</dt-radio>
```

### With Slotted Label and Description

```vue
<dt-radio value="optionValue">
  My Label
  <template #description>
    My Description
  </template>
</dt-radio>
```

### With Group

```vue
<dt-radio
  value="optionValue"
  name="groupName"
  label="My Label"
/>
```

### With Validation State

<h4 style={{ color: 'red' }}>Error</h4>

```vue
<dt-radio
  value="optionValue"
  :validationState="VALIDATION_MESSAGE_TYPES.ERROR"
/>
```

<h4 style={{ color: 'Goldenrod' }}>Warning</h4>

```vue
<dt-radio
  value="optionValue"
  :validationState="VALIDATION_MESSAGE_TYPES.WARNING"
/>
```

<h4 style={{ color: 'green' }}>Success</h4>

```vue
<dt-radio
  value="optionValue"
  :validationState="VALIDATION_MESSAGE_TYPES.SUCCESS"
/>
```

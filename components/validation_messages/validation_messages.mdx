# Validation Messages

<Subtitle>
  Validation messages are used to convey information to the user about the current state of the input element. These
  messages can have an error, warning or success type.
</Subtitle>

## Validation Message Format

Each provided validation message has the following format:

```json
{
  message: "Some informative message",
  type: "error|warning|success",
}
```

## Base Style

Validation messages are typically paired with an input element. They are currently built in to some of
our shared input components such as `dt-input`, `dt-radio-group`.

## Usage

### Import

```jsx

```

<h3 style={{ color: 'red' }}>With Error Messages</h3>

```vue

  :validationMessages="[{ message: 'Error', type: 'error' }]"

```

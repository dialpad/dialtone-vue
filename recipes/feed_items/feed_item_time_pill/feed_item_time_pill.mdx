# Time Pill

<Subtitle>
  Used to divide feed items that are from differente dates.
</Subtitle>

## Base Style

Pass in a display value as dateTimeDisplay, and the actual date into dateTime.

<b>Examples</b>:

- dateTimeDisplay: 'Today', 'Yesterday', 'Monday, September 13'

- dateTime: '2022-09-13'

## Usage

### Import

```jsx
import { DtRecipePagination } from '@dialpad/dialtone-vue';
```

### Basic

```html
<dt-recipe-pagination
  :total-pages="20"
/>
```

### With active-page

```html
<dt-recipe-pagination
  :total-pages="10"
  :active-page="5"
/>
```

### With max-visible

```html
<dt-recipe-pagination
  :total-pages="15"
  :max-visible="7"
/>
```

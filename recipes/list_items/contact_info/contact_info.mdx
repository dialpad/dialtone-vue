# DtRecipeContactInfo

<Subtitle>
  A contact information is an component that can be used to represent individual or group contact.
</Subtitle>

## Base Style

The contact information component has 4 slots that can be used for the most common use cases,
**header**, **subtitle**, **right** and **bottom** slot.

The **header** slot can be used to display html content or Vue component at header area.

The **subtitle** slot can be used to display content below the header slot. The slot has smaller
text size and lighter color than default slot.

The **right** slot works the same way, but its contents are placed to the right of the main slot.

The **bottom** slot can be used to display content below the subtitle slot.

Avatar is optional. When `avatarSrc` is specified, the avatar image will be displayed at the left.
If `avatarSrc` is empty, it will fallback to `avatarInitials`, and user initials will be displayed
in avatar.

## Usage

### Import

```jsx
import { DtRecipeContactInfo } from '@dialpad/dialtone-vue';
```

### Contact with avatar

```html
<dt-recipe-contact-info
  :avatar-src="avatarSrc"
  :user-status-color="userStatusColor"
>
  <template #header>
    <div class="d-fs16 d-fw-bold d-mr4">
      Joseph Lumaban
    </div>
  </template>
  <template #subtitle>
    <div class="d-d-flex d-ai-center">
      <div class="d-fs12 d-mt2">
        +1 (415) 123-4567
      </div>
      <icon-checkbox-filled
        class="d-fc-black-500 d-svg--size14 d-va-text-bottom"
      />
    </div>
  </template>
  <template #bottom>
    <div class="d-d-flex d-ai-center d-mtn6">
      <div class="d-w8 d-h8 d-mr4 d-bgc-magenta-200">
        &nbsp;
      </div>
      <div class="d-fs11 d-mr4">
        Aerolabs Support
      </div>
    </div>
  </template>
</dt-recipe-contact-info>
```

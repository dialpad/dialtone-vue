# DtRecipeContactInfo

<Subtitle>
  A contact information is an component that can be used to represent individual or group contact.
</Subtitle>

## Base Style

Contact information component has two main attributes: `display-name` and `phone-number`. If
`display-name` is specified, the `phone-number` will be displayed in subtitle area. Avatar is
optional. When `avatarSrc` is specified, the avatar image will be displayed. If `avatarSrc` is
empty, it will fallback to `avatarInitials`, and user initials will be displayed in avatar.
Group information is placed at the bottom.

## Variants

The contact information component has 4 slots that can be used for the most common use cases,
**header**, **subtitle**, **right** and **bottom** slot. All of the slots are optional.

The **header** slot can be used to display extra html content or Vue component at header area.

The **subtitle** slot can be used to display content below the header slot. The slot has smaller
text size and lighter color than default slot.

The **right** slot works the same way, but its contents are placed to the right of the main slot.

The **bottom** slot can be used to display content below the subtitle slot.

## Usage

### Import

```jsx
import { DtRecipeContactInfo } from '@dialpad/dialtone-vue';
```

### Contact with avatar

```html
<dt-recipe-contact-info
    displayName="Joseph Lumaban"
    avatarSrc="static/media/_/recipes/list_items/contact_info/avatar.png"
    userStatusColor="green"
    phoneNumber="+1 (415) 123-4567"
    showAttestation
/>
```

### Unknown contact (phone number only) with attestation

```html
<dt-recipe-contact-info
  :phone-number="phoneNumber"
  show-attestation
  @click="onClick"
>
  <template #subtitle>
    San Francisco, California
  </template>
</dt-recipe-contact-info>
```

### Unknown contact with "marked as spam" button

```html
<dt-recipe-contact-info
  phone-number="+1 (415) 123-4567"
  @click="onClick"
>
  <template #subtitle>
    <div
      class="d-fc-red-500"
      @click.stop="onMarkAsSpam"
    >
      Mark as spam
    </div>
  </template>
</dt-recipe-contact-info>
```

### Group contact

```html
<dt-recipe-contact-info
  :display-name="displayName"
  :phone-number="phoneNumber"
  :group-name="groupName"
  :group-color-class="groupColorClass"
  show-attestation
  @click="onClick"
/>
```
